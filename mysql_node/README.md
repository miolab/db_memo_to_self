# Usage MySQL with Node.js/ (Local)

- Env & Version

  ```
  $ mysql --version
  mysql  Ver 14.14 Distrib 5.7.29, for osx10.14 (x86_64) using  EditLine wrapper

  $ node --version
  v12.16.1

  $ npm --version
  6.14.4
  ```

  - [npm: mysql](https://www.npmjs.com/package/mysql)


## Initial Setting

```
$ mkdir mysql_node && cd mysql_node

$ npm install mysql

$ code app.js
.
.

$ node app.js
```
